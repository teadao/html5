CSS используется для дизайна страницы

Каскадные таблицы стилей (CSS) используются для форматирования макета веб-страницы. С помощью CSS вы можете контролировать цвет, шрифт, размер текста, расстояние между элементами и расположение элементов, то, какие фоновые изображения или цвета фона должны использоваться, настроить различное отображение для разных устройств и размеров экрана и т.д.
Внутренний CSS указывается в теге <head> внутри тега <style> после тега <body>
Встроенный CSS используется для применения уникального стиля к определенному элементу HTML и указывается в открывающем теге при помощи атрибута style="....."; все свойства стиля прописываются в ковычках через точку с запятой.

Свойство CSS border определяет границу вокруг элемента HTML


Свойство CSS padding определяет отступ (пробел) между текстом и рамкой


Свойство CSS margin определяет поле (пробел) за пределами границы


Еще бывает внешний CSS. На внешние таблицы стилей можно ссылаться с помощью полного URL-адреса или пути относительно текущей веб-страниц. Такие ссылки указываются в теге <head> при помощи тега <link> с атрибутами rel и herf. Примеры приведены ниже:

<link rel="stylesheet" href = "https : // www.w3schools.com / html / styles.css" / >

<link rel="stylesheet" href="/html/styles.css">

<link rel="stylesheet" href="styles.css">


Немного о ссылках

Самым важным атрибутом тега <a> является атрибут href, указывающий место назначения ссылки

Ссылка не обязательно должна быть текстом. Ссылка может быть изображением или любым другим элементом HTML

Атрибут title указывает дополнительную информацию об элементе. Информация отображается в виде текста, всплывающей подсказки, при наведении курсора мыши на элемент.

По умолчанию связанная страница откроется в текущем окне браузера. Чтобы изменить это, вы должны указать определенное свойство для атрибута target.

Если target = "_ blank", ссылка откроется в новом окне или вкладке браузера.Visit W3Schools!


_self- По умолчанию. Ссылка откроется в том же окне / вкладке.Visit W3Schools!

_parent - Открывает документ в родительском фрейме.Visit W3Schools!

_top - Открывает документ во всем окне.Visit W3Schools!
