таблицы HTML

Таблицы HTML позволяют веб-разработчикам размещать данные в строках и столбцах.

Определяет HTML-таблицу Тег table. Каждая строка таблицы определяется тегом tr. Каждый заголовок таблицы определяется тегом th. Каждые данные / ячейка таблицы определяются тегом td.
По умолчанию текст в элементах th выделен полужирным шрифтом и центрирован.
По умолчанию текст в элементах td обычный и выровнен по левому краю.
Firstname th элемент Age
td элемент td элемент
Eve Jackson 94

Примечание: td элементы являются контейнерами данных таблицы. Они могут содержать все виды HTML-элементов; текст, изображения, списки, другие таблицы и т. д.
