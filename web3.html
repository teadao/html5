таблицы HTML

Таблицы HTML позволяют веб-разработчикам размещать данные в строках и столбцах.

Определяет HTML-таблицу Тег table. Каждая строка таблицы определяется тегом tr. Каждый заголовок таблицы определяется тегом th. Каждые данные / ячейка таблицы определяются тегом td.
По умолчанию текст в элементах th выделен полужирным шрифтом и центрирован.
По умолчанию текст в элементах td обычный и выровнен по левому краю.
Чтобы добавить заголовок к таблице, используйте тег caption. Тег caption должен быть вставлен сразу после тега table.

Чтобы установить интервал границ для таблицы, используйте свойство CSS: border-spacing:5px
Чтобы сделать ячейку более одной строки(растянуть по вертикали), используйте атрибут:rowspan (th rowspan="3") и укажите нужное количество строк.
Чтобы ячейка занимала более одного столбца, используйте атрибут: colspan(th colspan="3") и укажите нужное количество столбцов над которыми должна распологатся строка.

Примечание: td элементы являются контейнерами данных таблицы. Они могут содержать все виды HTML-элементов; текст, изображения, списки, другие таблицы и т. д.

HTML списки

Неупорядоченный список HTML начинается с тега ul. Каждый элемент списка начинается с тега li. По умолчанию элементы списка будут отмечены маркерами (маленькими черными кружками). для выбора маркера в неупорядоченном списке элементов используется свойство CSS list-style-type. Свойство CSS list-style-type может иметь одно из следующих значений
disc Устанавливает маркер элемента списка в виде маркера (по умолчанию).
circle Устанавливает маркер элемента списка круг.
square Устанавливает маркер элемента списка в квадрат.
none Элементы списка не будут отмечены.

Упорядоченный список HTML начинается с тега ol. Каждый элемент списка начинается с тега li. По умолчанию элементы списка будут отмечены цифрами.
  1. Coffee
  2. Tea
  3. Milk

также HTML поддерживает списки описаний. Список описаний - это список терминов с описанием каждого термина. Тэг dl определяет список описания, тег dt определяет термин (имя), и тег dd описывает каждый термин
