таблицы HTML

Таблицы HTML позволяют веб-разработчикам размещать данные в строках и столбцах.

Определяет HTML-таблицу Тег table. Каждая строка таблицы определяется тегом tr. Каждый заголовок таблицы определяется тегом th. Каждые данные / ячейка таблицы определяются тегом td.
По умолчанию текст в элементах th выделен полужирным шрифтом и центрирован.
По умолчанию текст в элементах td обычный и выровнен по левому краю.
Чтобы добавить заголовок к таблице, используйте тег caption. Тег caption должен быть вставлен сразу после тега table.

Чтобы установить интервал границ для таблицы, используйте свойство CSS: border-spacing:5px
Чтобы сделать ячейку более одной строки(растянуть по вертикали), используйте атрибут:rowspan (th rowspan="3") и укажите нужное количество строк.
Чтобы ячейка занимала более одного столбца, используйте атрибут: colspan(th colspan="3") и укажите нужное количество столбцов над которыми должна распологаться строка.

HTML списки

Неупорядоченный список HTML начинается с тега ul. Каждый элемент списка начинается с тега li. По умолчанию элементы списка будут отмечены маркерами (маленькими черными кружками). для выбора маркера в неупорядоченном списке элементов используется свойство CSS list-style-type. Свойство CSS list-style-type может иметь одно из следующих значений
disc Устанавливает маркер элемента списка в виде маркера (по умолчанию).
circle Устанавливает маркер элемента списка круг.
square Устанавливает маркер элемента списка в квадрат.
none Элементы списка не будут отмечены.

Упорядоченный список HTML начинается с тега ol. Каждый элемент списка начинается с тега li. По умолчанию элементы списка будут отмечены цифрами. Упорядоченный список может быть числовым или алфавитным. Атрибут type для тега ol определяет тип маркера элемента списка. По умолчанию отсчет в упорядоченном списке начинается с 1. Если вы хотите начать отсчет с определенного числа, вы можете использовать атрибут: start.
type="1"Элементы списка будут пронумерованы цифрами (по умолчанию).
type="A" Пункты списка будут пронумерованы заглавными буквами.
type="a"Пункты списка будут пронумерованы строчными буквами.
type="I"Элементы списка будут пронумерованы римскими цифрами в верхнем регистре.
type="i"Элементы списка будут пронумерованы римскими цифрами в нижнем регистре.

Списки HTML можно стилизовать разными способами с помощью CSS.
Один из популярных способов - стилизовать список по горизонтали, чтобы создать меню навигации.

  • Используйте list-style-type свойство CSS, чтобы определить маркер элемента списка.
  • Используйте li элемент HTML для определения элемента списка
  • Списки могут быть вложенными

  • Используйте свойство CSS float:left or right для отображения списка по горизонтали

HTML- блоки и встроенные элементы

Элемент уровня блока всегда начинается с новой строки и занимает всю доступную ширину (тянется влево и вправо, насколько это возможно).

Вот элементы блочного уровня в HTML: address; article; aside; blockquote; canvas; dd; div; dl; dt; fieldset; figcaption; figure; footer; form; h1; h6; header; hr; li; main; nav; noscript; ol; p; pre; section; table; tfoot; ul; video;

Встроенный элемент не начинается с новой строки и занимает ровно столько ширины, сколько необходимо.

Вот встроенные элементы в HTML: a; abbr; acronym; b; bdo; big; br; button; cite; code; dfn; em; i; img; input; kbd; label; map; object; output; q; samp; script; select; small; span; strong; sub; sup; textarea; time; tt; var;
Элемент span - это встроенный контейнер, используемый для разметки части текста или части документа.
Элемент span не имеет обязательных атрибутов, стиль, класс и идентификатор применяемые к элементу HTML являются общими и применяются ко всем дочерним элементам.
При использовании вместе с CSS элемент span можно использовать для стилизации частей текста HTML документа.

Использование атрибута Class

Атрибут HTML class используется для указания класса для элемента HTML. Несколько элементов HTML могут использовать один и тот же имя атрибута класс.
Атрибут class можно присвоить как блочному так и встроенному элементу HTML.
Элементы HTML могут принадлежать более чем к одному классу. Чтобы определить несколько классов, разделите имена классов пробелом, например div class="имя1 имя2">. Элемент будет стилизован в соответствии со всеми указанными классами.

class атрибут может быть использован на любом HTML - элементе. Имя class чувствительно к регистру!
Создать класс: напишите символ точки (.) за которым следует имя класса. Затем определите свойства CSS в фигурных скобках {.....}.
Имя класса также может использоваться JavaScript для выполнения определенных задач для определенных элементов. JavaScript может получить доступ к элементам с определенным именем класса с помощью getElementsByClassName()метода.


HTML Атрибут id

Атрибут id определяет уникальный идентификатор для HTML - элемента. Значение id атрибута должно быть уникальным (не повтаряющемся) в документе HTML. id Атрибут используется для указания ссылки к определенному стилю в таблице стилей. Он также используется JavaScript для доступа и управления элементми с определенным идентификатором. синтаксис id: напишите символ решетки (#) за которым следует имя идентификатора. Затем определите свойства CSS в фигурных скобках {.....}.
В коде у нас есть h2 элемент c именем идентификатора «myHeader». Этот h2 элемент будет оформлен в соответствии с #myHeader определением стиля в разделе style данного документа HTML: оформлен заголовок данного раздела.
