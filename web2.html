HTML изображения


HTML Тег img используется для вставки изображения на веб-страницу. Изображения технически не вставляются на веб-страницу; изображения связаны с веб-страницами.
img Тег создает удерживающее пространство для ссылочного изображения. img Тег пуст, он содержит только атрибуты и не имеет закрывающего тега.

img Тег имеет два обязательных атрибута:

src - указывает путь к изображению

alt - Задает альтернативный текст для изображения.


Карта изображений

Тег HTML map и атрибут usemap=#(добовляется для присвоения id каринке) определяет карту изображений. Карта изображений - это изображение с интерактивными областями. Области определяются (выделяются) одним или несколькими тегами area.
для добавления интерактивной картинки к тегу img добовляем атрибут #usemap и указываем имя на которое будем ссылаться в теге map после атрибта name

для добавления интерактивных облостей на картинку, нужно указать форму (shape) интерактивной области и выбрать одно из следующих свойств (значений):
rect - определяет прямоугольную область
circle - определяет круговую область
poly - определяет многоугольную область
default - определяет весь регион
также надо определить координаты облости выделения и ее размер. значения poly (определяет многоугольную область)

Координаты для значения rect приходят парами, одна для оси x, другая для оси y. Итак: координаты 34,44 расположены на 34 пикселя от левой стороны и на 44 пикселя сверху и Координаты 270,350 расположены на 270 пикселей от левой стороны и 350 пикселей сверху, теперь у нас достаточно данных, чтобы создать интерактивную прямоугольную область: которая выделяет компьютер.
shape="rekt"
Чтобы добавить область круга, сначала найдите координаты центра круга, затем укажите радиус круга
Форма = "poly" Он shape="poly"содержит несколько координатных точек, что создает форму, образованную прямыми линиями многоугольник (круассана).Это можно использовать для создания любой формы. Координаты идут парами, одна для оси x, другая для оси y.
нужно найти координаты x и y для всех краев круассана

Можно указать фоновые изображение почти для любого видимого элемента HTML. В этом примере фоновое изображение указывается для элемента div.
Чтобы добавить фоновое изображение в элемент HTML, используйте styleатрибут HTML и background-imageсвойство CSS
также можно указать фоновое изображение теге head через тег style указав нужный HTML элемент с использованием атрбута background-image: url('ссылка на картинку').
Если вы хотите, чтобы на всей странице было фоновое изображение, вы должны указать фоновое изображение для элемента.
Повтор фона.Если фоновое изображение меньше размера элемента, изображение будет повторяться по горизонтали и вертикали, пока не достигнет конца элемента.
Чтобы фоновое изображение не повторялось, установите для background-repeatсвойства значение no-repeat.
Если вы хотите, чтобы фоновое изображение покрыло весь элемент, то можете установить для свойства background-size значение cover. Чтобы убедиться, что весь элемент всегда покрыт, установите для свойства background-attachment значениеfixed. Таким образом, фоновое изображение будет покрывать весь элемент без растяжения (изображение сохранит свои исходные пропорции)
Если вы хотите, чтобы фоновое изображение растянулось, чтобы соответствовать всему элементу, вы можете установить для background-sizeсвойства значение 100% 100%

HTML Тег picture

HTML Элемент picture позволяет отображать разные изображения для разных устройств или размеров экрана.
HTML Элемент picture дает веб-разработчикам больше гибкости при указании ресурсов изображений. Элемент picture содержит один или более элементов source, каждый из которых имеет ссылку на различные изображения через атрибут srcset. Таким образом, браузер может выбрать изображение, которое лучше всего соответствует текущему виду и / или устройству. У каждого элемента есть media атрибут, который определяет, когда изображение является наиболее подходящим.
Примечание. Всегда указывайте элемент img как последний дочерний элемент picture элемента. Элемент img используется в браузерах, которые не поддерживают элемент picture, или если ни один из source тегов не подходит.

Пропускная способность. Если у вас маленький экран или устройство, то нет необходимости загружать большой файл изображения. Браузер будет использовать первый source элемент с совпадающими значениями атрибутов и игнорировать любой из следующих элементов.
​
Поддержка формата. Некоторые браузеры или устройства могут поддерживать все форматы изображений. Используя picture элемент, вы можете добавить одно или несколко изображений в разных(во всех) форматах, и браузер будет использовать первый формат, который он распознает, и игнорировать любой из следующих элементов.

Пример

picture
source srcset="img_avatar.png"
source srcset="img_avatar.jpg"
img src="img_avatar" alt="Avatar" style="width:auto;"
picture
