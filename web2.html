HTML изображения


HTML Тег <img> используется для вставки изображения на веб-страницу. Изображения технически не вставляются на веб-страницы, изображения связаны с веб-страницами.
Тег <img> создает удерживающее пространство для изображения. Тег <img> содержит только атрибуты и не имеет закрывающего тега.

Тег <img> имеет два обязательных атрибута:

атрибут src - указывает путь к изображению

атрибут alt - Задает альтернативный текст для изображения.


Карта изображений

HTML Тег <map> и атрибут usemap=#(добовляется для присвоения id каринке) определяет карту изображений. Карта изображений - это изображение с интерактивными областями. Области определяются (выделяются) одним или несколькими тегами <area>.
для добавления интерактивной картинки к тегу <img> добовляем атрибут #usemap и указываем имя на которое будем ссылаться в теге <map> после атрибта name

для добавления интерактивных облостей на картинку, нужно указать атрибут shape для тега <area> и установть одно из следующих свойств (значений):
rect - определяет прямоугольную область
circle - определяет круговую область
poly - определяет многоугольную область
default - определяет картинку целиком
также надо определить координаты облости выделения и ее размер.

Координаты для значения rect приходят парами, одна для оси x, другая для оси y. Координаты 34,44 расположены на 34 пикселя от левой стороны и на 44 пикселя сверху и Координаты 270,350 расположены на 270 пикселей от левой стороны и 350 пикселей сверху, теперь у нас достаточно данных, чтобы создать интерактивную прямоугольную область, которая выделяет компьютер: <area shape="rekt" coords="34,44,270,350" alt="Computer" href="https://www.google.com/">
Чтобы добавить область круга, сначала найдите координаты центра круга, затем укажите радиус круга для создания интерактивной области, которая выделяет чашку кофе.

Форма = "poly" содержит несколько координатных точек, что создает форму, образованную прямыми линиями многоугольник (круассана). Это можно использовать для создания любой формы. Координаты идут парами, одна для оси x, другая для оси y. нужно найти координаты x и y для всех краев круассана


Интерактивная область также может запускать функцию JavaScript. Добавьте свойство (значение) clic к тегу area для запуска функции JavaScript. На картинке ниже мы используем атрибут onclick для выполнения функции JavaScript при щелчке по области кофе.

Можно указать фоновые изображение почти для любого видимого элемента HTML. В этом примере фоновое изображение указывается для элемента <div>.
Чтобы добавить фоновое изображение в элемент HTML используйте атрибут style и свойство CSS background-image.
также можно указать фоновое изображение в теге <head> через тег <style> указав нужный HTML элемент с использованием атрбута background-image: и url('ссылка на картинку').
Если вы хотите, чтобы на всей странице было фоновое изображение, нужно указать фоновое изображение для элемента <body>.

Повтор фона.

Если фоновое изображение меньше размера элемента, изображение будет повторяться по горизонтали и вертикали, пока не достигнет конца элемента.
Чтобы фоновое изображение не повторялось, установите для свойства background-repeat значение no-repeat.
Если вы хотите, чтобы фоновое изображение покрыло весь элемент, то можете установить для свойства background-size значение cover. Чтобы убедиться, что весь элемент всегда покрыт, установите для свойства background-attachment значение fixed. Таким образом, фоновое изображение будет покрывать весь элемент без растяжения (изображение сохранит свои исходные пропорции)
Если вы хотите, чтобы фоновое изображение растянулось, чтобы соответствовать всему элементу, можете установить для свойства background-size значение 100% 100%

HTML Тег <picture>

HTML Тег <picture> позволяет отображать разные изображения для разных устройств или размеров экрана.
HTML Тег <picture> дает веб-разработчикам больше гибкости при указании ресурсов изображений. <picture> содержит один или более элементов <source>, каждый из которых имеет ссылку на различные изображения через атрибут srcset. Таким образом, браузер может выбрать изображение, которое лучше всего соответствует текущему виду и / или устройству. У каждого <source> элемента есть media атрибут, который определяет, когда изображение является наиболее подходящим.
Примечание. Всегда указывайте элемент <img> как последний дочерний элемент <picture> элемента. Элемент <img> используется в браузерах, которые не поддерживают элемент picture, или если ни один из source тегов не подходит.

Если у пользователя маленький экран или устройство, то нет необходимости загружать большой файл изображения. Браузер будет использовать первый <source> элемент с совпадающими значениями атрибутов и игнорировать любой из следующих элементов.
​
Некоторые браузеры или устройства могут поддерживать не все форматы изображений. Используя тег <picture/>, вы можете добавить одно или несколко изображений в разных(во всех) форматах, и браузер будет использовать первый формат, который он распознает, и игнорировать любой из следующих элементов.

Пример

<picture>
<source srcset="img_avatar.png">
<source srcset="img_avatar.jpg">
<img src="img_avatar" alt="Avatar" style="width:auto;">
<picture/>
